User Authentication and Authorization with Bearer Token

 Description

This is a Node.js backend application built with Express.js, MongoDB (Mongoose), and JSON Web Tokens (JWT) that performs secure user authentication and authorization.
It features register, login, JWT protected routes, and EJS based UI for simple user interaction.

 Features

MVC-structured backend using Express.js

User registration with hashed passwords

User login with JWT token generation

Protected routes using middleware to verify JWT

Browser UI using EJS templates for Register, Login & Profile

Cookie-based token storage for session persistence

Logout functionality

MongoDB Atlas integration

Fully deployed and working app
